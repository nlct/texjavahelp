The test files assume the installation described in the [`INSTALL.md`](../../INSTALL.md)
file. I have symlinks in the TEXMF tree. The alternative is to copy
over the `.jar`, `.tlu`, `.sh`, `.def` and `.pdf` files whenever they are updated.

The `--epub` option is still under development and isn't fully
functioning.

The `test-helpset.tjh` file can be viewed with:
```
tjhviewer test-helpset.tjh
```
